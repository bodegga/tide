# Tide Gateway v1.1.0 - ARM64 Tor Gateway

## What This Is

Universal OVA package containing Alpine Linux ARM64 - ready to import into any hypervisor.

**File**: `tide-gateway.ova` (168MB)

## Import Instructions

### Any Hypervisor:
1. Download `tide-gateway.ova`
2. File → Import/Open
3. Select the OVA
4. Boot the VM

### Supported:
- ✅ UTM (Mac ARM64)
- ✅ Parallels Desktop (Mac ARM64)
- ✅ VMware Fusion (Mac ARM64)
- ✅ VirtualBox (ARM64)
- ✅ QEMU/KVM (ARM64 servers)

## What's Inside

- Alpine Linux 3.19 ARM64
- 168MB disk image
- EFI bootable
- 2 network adapters (Shared + Host-Only)

## Current Status

⚠️ This OVA boots Alpine Linux but **Tor is not yet configured**.

The Tor gateway setup needs to be completed after import.
